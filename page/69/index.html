与深圳音乐厅的第一次亲密接触

发表于 2010-04-04 更新于 2025-08-09 阅读次数： Waline： 阅读次数： 本文字数： 710 阅读时长 ≈ 1 分钟

By Long Luo

2010/03/28欣赏了陈萨的肖邦钢琴音乐会，2010/04/04下午欣赏了深圳高级中学的百合合唱团带来的音乐会，感慨颇多，在此限于时间和精力，仅简单记录一下自己的感想：

  1. 有才能和有内涵才是根本啊！做一个有才有内涵的人是我努力的方向。

  2. 钢琴不愧是乐器之王！

  3. 我终于知道什么是世界级水平的钢琴演奏了！

  4. 被音乐会主持人浑厚富有磁性的男中音被迷住了，外加这次的3位新主持！英雄出少年啊！对比自己，深感惭愧！

  5. 深圳音乐厅的服务员个个都是美女，而且是气质才能俱佳的那种！

  6. 什么是好的声音？

  7. 我的手脚太笨尤其是手指又笨又短，看来今生要与很多乐器无缘了，希望自己经过锻炼让手指灵活一点！

四月，我们看海去

作者：艺衡

四月，我们看海去。

夏天说早了，春天说迟了，然而，我们选择四月。

是的，报春的花朵将于四月从枝头坠落。

是的，初夏刚刚露面，还热得不够气氛，海滩也许是孤寂的，然而，我们选择四月。

因为，四月是变化的季节。

默默的春潮开始鼓动起夏的浪花，强大的暖流正铺天盖地砸向礁石，涌入海滩，春叶的幼稚转入夏的成熟，春的播种滋生着夏的繁荣。

我们喜欢这热闹前的寂寞，我们喜欢这幼稚向成熟的变化，我们看海去！

不必等到滚滚的人流都涌向海滩，你再去看海，海的壮阔，不取决于岸边的喧闹。

不必叹息你没有跟上春的脚步，四月的海是大自然的第二个春天，你在这个时候看海去，你便是另一个意义上的弄潮儿。

我们看海去，看变化的海，早己是不早，但，迟也不迟！

抓住人生中的四月吧，将悲春的叹息留给残花，走向大海，迎接火热的五月、六月。

不管天时，四月，我们看海去！

By Long Luo@April 4th, 2010, at Futian, Shenzhen, China.

在蓝天和白云的亲密接触---记第一次飞机旅行

发表于 2010-02-22 更新于 2025-05-02 阅读次数： Waline： 阅读次数： 本文字数： 2.9k 阅读时长 ≈ 3 分钟

By Long Luo

一、离家

2010年02月21日是年假的最后一天了，我要和妈妈的美食，早睡早起说bye-bye了，心中虽然有些不舍。自己从正式步入社会成为一名上班族，一晃就7个月过去了。之前原本希望自己在过年的时候对自己这几个月的工作和生活进行一番总结，但是由于自己的种种懒惰的原因，自己都未能全部完成，心里稍微有点失落和自责。

昨天晚上收拾好了东西之后很早就睡觉了，因为第二天要早起。早上6:30左右就听见爸爸妈妈在楼下忙了，我赶紧爬起来，穿上衣服，打开手机看一下才6:27。起床之后把自己收拾好的东西提下楼，锻炼了5分钟左右，好好吃了一餐丰盛的早餐。

吃晚饭后，我又一次像以前一样看着自己面前的房子，对他说声bye-bye，自己有生之年这样的时刻还会持续很多次，我想：当我自己再一次重逢的话，我一定会变得更强大！

Auf Wiedersehen! Mein House!

二、上车

离开了家，来到了集镇上，爸爸帮我买了一箱本市的特产—冻米糖，我买了一小袋香蕉，毕竟机场的东西太贵，而航班还在下午3点。爸爸送我上了车，我又一次坐在靠窗口的位置，我看着爸爸，心里不禁一阵发酸。这个画面，多次在我高中，大学的时候重演，但没有一次相今天这样让我难过。想自己12岁就离家，住在学校里，我都一路走过来，自己15岁就到离家150公里求学，每学期回一次家，都没有今天这般难过。爸爸明显已经老了，50多岁的人了，几根白发那么明显，我虽然现在已经进入社会了，但是爸爸妈妈还是像往常一样操心我的事，只怪自己没有能力，年纪又小，如果我不是这样，也许我爸爸妈妈要少受点苦吧？

车子发动了，我想我改变不了很多事，但我希望自己以后的工作地点可以离爸爸妈妈更近一点……

三、昌北机场

到了市里的客运站，买了一张到南昌火车站的票，一会儿就上车了。坐在车上已经可以感受到车窗外明媚的阳光，周围的温度也逐渐上升。快要到南昌城里，堵车变得很严重。车子走走停停，花了2个多小时终于到了火车站。下车之后，问了一下街上的巡警，得知到站前路去坐车。原本以为车站附近应该有公交去昌北机场的，结果证明南昌不如成都和深圳，没有公交。于是我只有向前走了大概150米，走到了乘坐机场大巴的地点，花了10块钱，上了车。

上午11点的时候，大巴准时出发。但是一路上我发现大巴停靠了很多站。车上最后变得很拥挤。我们一路上经过了省政府，和雄伟的八一大桥，然后车就上了高速。走了20分钟，看到了正在大兴土木的建筑，不过已经可以看出了新航站楼了。

到了航站楼里了，先左右看了下，找个位置休息了一下，发现自己来的太早了。先去托运一下自己的行李，和美丽的服务员说自己要观景好的，靠窗的座位，行李为9.56Kg，托运好了之后拿到自己的登机牌，位置为后排19A。于是自己一身轻地走出把航站楼的楼上楼下，左左右右认真看了一下。然后走出航站楼，花了半个小时，看了一下周围的情况。

前前后后溜达了一个小时，然后决定去内场。首先要接受拍照留底，然后过了之后，走到里面。看见了Smith Aero生产的X光检查设备，把自己的包放上去之后，然后通过一到安检门，之后工作人员拿着检测仪器，像审讯犯人一样把我从头到脚，衣服里里里外外检测了个遍。

严密的安检结束了，我终于来到了内场。虽然自己之前曾经在成都送一位同学到成都双流国际机场，但是一直没有机会去内场看下飞机，现在终于有这个机会了，我非常兴奋。我走到窗户玻璃边，首先看见的几架中小型的飞机，不是737系列就是320系列，大部分都是东航的班机。东方航空的A320飞机，与737相比主要区别是机翼尾部的翼尖小翼。昌北机场实在是个小机场，都没有什么大型飞机，我想自己几年之内肯定有机会去北京或者上海的机场，到时候肯定可以让自己大饱眼福。我会为之努力的，trust me！

三、起飞

登上飞机之后看见了美丽的空姐，

飞机依靠引擎低转速的推力慢慢地移动到跑道上，然后飞机广播说，飞机准备起飞了，之后我感觉到引擎开始轰鸣起来，自己被一股力往后推，显然飞机在开始加速了，飞机越飞越快，但是从窗外看却不明显。机窗玻璃应该是特制的减速玻璃。几秒种后之后我感觉到飞机已经拔地而起了，之后头朝上开始爬升。

从窗户往外看，地下的建筑物显得越来越小，看见了赣江及来机场经过的八一大桥。

这是一次未知的远航----写在毕业四个月之际

发表于 2009-10-31 更新于 2025-05-03 阅读次数： Waline： 阅读次数： 本文字数： 644 阅读时长 ≈ 1 分钟

By Long Luo

时间已经到了10月底了，按照农历来算已经快要到冬天了，而在深圳却感觉不到深秋的寒意。这个时候在我老家和成都的话，应该到处是黄叶纷纷落下的场景和空气中弥漫的成熟的气息。走在深圳的街头，炙热的阳光和满眼的绿色，俨然是夏天，只有小区里垂下来的一串串芒果，才提醒我原来秋天已经来到了深圳。

持续了几周没有降雨，这在深圳是很少见的。记得来了深圳之后经常是凌晨和傍晚会下一场小雨，把空气洗的格外清新。而现在的我就如同这天气一样对前途和未来充满了焦灼不安。离开了学校这个温暖的港湾到踏入社会这个充满了未知风暴及暗礁的海域，既有战胜未知的渴望，也有彷徨和迷茫。学生生涯时也曾想早日泛舟社会这个大海，而在大海里几个月之后才发现自己这条船在锻造的时候没有锻造出足够的抗击风浪的能力。回首自己走过的这条路，不知道自己选择的这条航线是否能够能够到达鲜花和荣耀的彼岸？

“穿过幽暗地岁月，也曾感到彷徨，当你低头地瞬间，才发觉脚下的路！”不知道为什么，工作之后很喜欢许巍的歌，喜欢他略带沙哑的嗓音，那来自灵魂的歌唱。想起十几年前的斑驳的上学路上的一个小孩，在想着自己将要去何方，这个世界到底有多大，自己多想看看这个世界的各个角落。正如许巍的《曾经的你》，“曾梦想仗剑走天涯，看一看世界的繁华，年少的心总有些轻狂，如今你四海为家！”

这是一次未知的远航，狂风会带来巨浪，但也会让风帆鼓起；风浪再大，只会锤炼水手的坚强。远处那漆黑的天幕，依稀可以看到星星的亮光！

Long Luo 2009年10月31日下午于深圳
