快速数论变换(Number Theoretic Transform)

发表于 2022-05-01 更新于 2023-11-14 分类于 Data Structures and Algorithms Waline： 阅读次数： 本文字数： 4.1k 阅读时长 ≈ 4 分钟

By Long Luo

之前的文章 快速傅里叶变换(FFT)算法 和 快速傅里叶变换(FFT)算法的实现及优化 详细介绍了 FFT\textit{FFT}FFT 的具体实现及其实现。

FFT\textit{FFT}FFT 优点很多，但缺点也很明显。例如单位复根的实部和虚部分别是一个正弦及余弦函数，有大量浮点数计算，计算量很大，而且浮点数运算产生的误差会比较大。

如果我们操作的对象都是整数的话，其实数学家已经发现了一个更好的方法：快速数论变换 (Number Theoretic Transform)\textit{(Number Theoretic Transform)}(Number Theoretic Transform) [1]。

快速数论变换(NTT)

FFT\textit{FFT}FFT 的本质是什么？

  • 将卷积操作变成了乘法操作。

是什么让 FFT\textit{FFT}FFT 做到了 O(nlogn)O(nlogn)O(nlogn) 的复杂度？

  • 是单位复根。

那有没有什么其他的东西也拥有单位根的这些性质呢？

答案当然是有的，原根[2]就具有和单位根一样的性质。

所以快速数论变换 NTT\textit{NTT}NTT 就是以数论为基础的具有循环卷积性质的，用有限域上的单位根来取代复平面上的单位根的 FFT\textit{FFT}FFT。

原根

仿照单位复数根的形式，也将原根的取值看成一个圆，不过这个圆上只有有限个点，每个点表达的是模数的剩余系中的值。

在 FFT\textit{FFT}FFT 中，我们总共用到了单位复根的这些性质：

  1. nnn 个单位复根互不相同；
  2. ωnk=ω2n2k\omega_n^k=\omega_{2n}^{2k}ωnk​=ω2n2k​；
  3. ωnk=−ωnk+n/2\omega_n^{k}=-\omega_n^{k+n/2}ωnk​=−ωnk+n/2​；
  4. ωna×ωnb=ωna+b\omega_n^a\times\omega_n^b=\omega_n^{a+b}ωna​×ωnb​=ωna+b​。

我们发现原根具有和单位复根一样的性质，简单证明[3]：

令 nnn 为大于 111 的 222 的幂，ppp 为素数且 n∣(p−1)n \mid (p-1)n∣(p−1)，ggg 为 ppp 的一个原根。

我们设 gn=gp−1ng_n=g^{\frac{p-1}{n}}gn​=gnp−1​：

  1. gnn=gn⋅p−1n=gp−1g_n^n=g^{n\cdot\frac{p-1}{n}}=g^{p-1}gnn​=gn⋅np−1​=gp−1

  2. gnn2=gp−12g_n^{\frac{n}{2}}=g^{\frac{p-1}{2}}gn2n​​=g2p−1​

  3. ganak=gak(p−1)an=gk(p−1)n=gnkg_{an}^{ak}=g^{\frac{ak(p-1)}{an}}=g^{\frac{k(p-1)}{n}}=g_n^kganak​=ganak(p−1)​=gnk(p−1)​=gnk​

显然

  1. gnn≡1(modp)g_n^n \equiv 1 \pmod pgnn​≡1(modp)

  2. gnn2≡−1(modp)g_n^{\frac{n}{2}} \equiv -1 \pmod pgn2n​​≡−1(modp)

  3. (gnk+n2)2=gn2k+n≡gn2k(modp)(g_n^{k+\frac{n}{2}})^2=g_n^{2k+n} \equiv g_n^{2k} \pmod p(gnk+2n​​)2=gn2k+n​≡gn2k​(modp)

所以将 gnkg_n^kgnk​ 和 gnk+n2g_n^{k+\frac{n}2}gnk+2n​​ 带入本质上和将 ωnk\omega_n^{k}ωnk​ 和 ωnk+n2\omega_n^{k+\frac{n}{2}}ωnk+2n​​ 带入的操作无异。

利用Vandermonde矩阵性质，类似 NTT\textit{NTT}NTT 那样，我们可以从 NTT\textit{NTT}NTT 变换得到逆变换 INTT\textit{INTT}INTT 变换，设 x(n)x(n)x(n) 为整数序列，则有：

NTT\textit{NTT}NTT : X(m)=∑i=0Nx(n)amn(modM)X(m) = \sum\limits_{i=0}^{N}x(n)a^{mn} \pmod MX(m)=i=0∑N​x(n)amn(modM)

INTT\textit{INTT}INTT : X(m)=N−1∑i=0Nx(n)a−mn(modM)X(m) = N^{-1}\sum\limits_{i=0}^{N}x(n)a^{-mn} \pmod MX(m)=N−1i=0∑N​x(n)a−mn(modM)

这里 N−1N^{-1}N−1，a−mn(modM)a^{-mn} \pmod Ma−mn(modM) 为模意义下的乘法逆元。

当然， NTT\textit{NTT}NTT 也是有自己的缺点的：比如不能够处理小数的情况，以及不能够处理没有模数的情况。对于模数的选取也有一定的要求，首先是必须要有原根，其次是必须要是 222 的较高幂次的倍数。
