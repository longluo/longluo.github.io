[Leetcode][1385. 两个数组间的距离值] 2种方法：模拟 和 二分搜索

发表于 2022-03-31 更新于 2023-11-14 分类于 Data Structures and Algorithms Waline： 阅读次数： 本文字数： 1.7k 阅读时长 ≈ 2 分钟

By Long Luo

这篇文章是 Leetcode 1385. 两个数组间的距离值 的 力扣社区题解 。

方法一：模拟

对于 arr1arr1arr1 数组中的每一个元素 xxx，枚举数组 arr2arr2arr2 中的每一个元素 yyy，检查是否对于每一个 yyy 都有 ∣x−y∣>d|x - y| > d∣x−y∣>d，如果是，就将答案增加 111。

复杂度分析

  • 时间复杂度：O(mn)O(mn)O(mn)，其中 arr1arr1arr1 中元素个数为 mmm，arr2arr2arr2 中元素个数为 nnn。

  • 空间复杂度：O(1)O(1)O(1)。

方法二： 二分搜索

在方法一中，要知道是否每一个 yyy 是不是满足 ∣x−y∣>d|x - y| > d∣x−y∣>d，我们枚举了所有 yyy。

实际上因为 d>=0d >= 0d>=0，假如arr2存在某个 yyy 满足 x−d≤y≤x+dx - d \le y \le x + dx−d≤y≤x+d，那么arr1中的数 xxx 就不满足要求。

先对arr2进行排序，之后枚举arr1每个元素 xxx，利用二分搜索来判断 xxx 是否满足要求。

复杂度分析

  • 时间复杂度：O((n+m)logm)O((n+m)logm)O((n+m)logm)，给 arr2 排序的时间代价是 O(mlog⁡m)O(m \log m)O(mlogm)，对于arr1中的每个元素都在arr2中二分的时间代价是 O(nlog⁡m)O(n \log m)O(nlogm)，故总时间复杂度为 O((n+m)logm)O((n+m)logm)O((n+m)logm)。

  • 空间复杂度：O(1)O(1)O(1)。
