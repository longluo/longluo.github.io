解析RSA加解密算法

发表于 2013-09-24 更新于 2023-11-14 分类于 Data Structures and Algorithms Waline： 阅读次数： 本文字数： 5.1k 阅读时长 ≈ 5 分钟

By Long Luo

一、RSA 是什么？

RSA\textit{RSA}RSA [1]公钥加密算法是1977年由Ron Rivest、Adi Shamirh和Leonard Adleman在（美国麻省理工学院）开发的。RSA\textit{RSA}RSA 取名来自开发他们三者的名字。

RSA\textit{RSA}RSA 算法基于一个十分简单的数论事实：将两个大素数相乘十分容易，但想要对其乘积进行因式分解却极其困难，因此可以将乘积公开作为加密密钥。

二、RSA算法原理

RSA\textit{RSA}RSA 算法是一种非对称密码算法，所谓非对称，就是指该算法需要一对密钥，使用其中一个加密，则需要用另一个才能解密。

下面我们就详细讲解下 RSA\textit{RSA}RSA 算法加解密过程。

2.1 RSA算法组成部分

  • 原文(Message)：需要加密的信息，可以是数字、文字、视频、音频等，用 MMM 表示。
  • 密文(Ciphertext)：加密后得到的信息，用 CCC 表示。
  • 公钥(Public Key)和私钥(Private Key)：用 PUPUPU 和 PRPRPR 表示。
  • 加密算法(Encryption)：若 E(x)E(x)E(x) 为加密算法，加密过程可以理解为 C=E(M)C = E(M)C=E(M) 根据原文和加密算法得到密文。
  • 解密算法(Decryption)：若 D(x)D(x)D(x) 为解密算法，解密过程可以理解为 M=D(C)M=D(C)M=D(C) 根据密文和解密算法得到原文。

2.2 RSA算法加密过程

  1. 随机选择两个不相同的素数 p,qp, qp,q；
  2. 将 p,qp, qp,q 相乘，记为 n=p×qn = p \times qn=p×q；
  3. 计算 nnn 的欧拉函数 φ(n)\varphi(n)φ(n) ，欧拉函数[2] 证明，当 p,qp, qp,q 为不相同的素数时，φ(n)=(p−1)(q−1)\varphi(n) = (p-1)(q-1)φ(n)=(p−1)(q−1)。
  4. 随机选择一个整数 eee，满足两个条件：φ(n)\varphi(n)φ(n) 与 eee 互质，且 1<e<φ(n)1 < e < \varphi(n)1<e<φ(n)。
  5. 计算 eee 对于 φ(n)\varphi(n)φ(n) 的模反元素 ddd，也就是说找到一个 ddd 满足 ed≡1mod  φ(n)ed \equiv 1 \mod \varphi(n)ed≡1modφ(n)。这个式子等价于 ed−1=k×φ(n)ed - 1 = k \times \varphi(n)ed−1=k×φ(n)，实际上就是对于方程 ed−k×φ(n)=1ed - k \times \varphi(n) = 1ed−k×φ(n)=1 求 (d,k)(d,k)(d,k) 的整数解。这个方程可以用扩展欧几里得算法[3]求解。
  6. 最终把 (e,n)(e,n)(e,n) 封装成公钥，(d,n)(d,n)(d,n) 封装成私钥。

由于理论太枯燥，我们来举个实际例子运行一遍这个算法。

  1. 随机选择两个不相同的素数 p,qp,qp,q。我们选择 p=103,q=97p=103, q=97p=103,q=97。
  2. 将 p,qp,qp,q 相乘，n=103×97=9991n=103 \times 97 = 9991n=103×97=9991
  3. φ(n)=(103−1)(97−1)=9792\varphi(n)=(103-1)(97-1)=9792φ(n)=(103−1)(97−1)=9792
  4. 随机选择一个 e=1213e = 1213e=1213，满足 φ(n)\varphi(n)φ(n) 与 eee 互质且 1<e<φ(n)1 < e < \varphi(n)1<e<φ(n)。
  5. 计算 eee 对于 φ(n)\varphi(n)φ(n) 的模反元素 ddd，即代入方程 ed−k×φ(n)=1ed - k \times \varphi(n)=1ed−k×φ(n)=1 求整数解。将 e=1213,φ(n)=9792e=1213,\varphi(n)=9792e=1213,φ(n)=9792 代入方程，得到 1213×d−9792×k=11213 \times d - 9792 \times k = 11213×d−9792×k=1，很容易可以找到 (k,d)(k,d)(k,d) 的一组解 k=510,d=4117k=510, d=4117k=510,d=4117。
  6. 封装公钥和私钥，最终得到的公钥 (e,n)=(1213,9991)(e,n)=(1213, 9991)(e,n)=(1213,9991)，私钥 (d,n)=(4117,9991)(d,n)=(4117, 9991)(d,n)=(4117,9991)。
    至此为止，我们有了原文 MMM ，公钥 (e,n)(e,n)(e,n) 和私钥 (d,n)(d,n)(d,n)。有了这些信息就可以开始加密和解密了。

2.2.1 加密

在 RSA\textit{RSA}RSA 算法中，加密过程实际上就是利用公钥 (e,n)(e,n)(e,n) 计算 C=Me(mod n)C = M^e (mod \ n)C=Me(mod n)。假设原文 M=6M=6M=6，代入上面的值，得到 C=61213(mod 9991)=7863C = 6^{1213} (mod \ 9991) = 7863C=61213(mod 9991)=7863。
于是 C=7863C=7863C=7863，Alice就把 786378637863 发给了Bob。

2.2.2 解密

Bob收到了密文 C=7863C=7863C=7863，就用自己的私钥 (d,n)=(4117,9991)(d,n)=(4117, 9991)(d,n)=(4117,9991) 进行解密。RSA\textit{RSA}RSA 证明，原文的 eee 次方对 nnn 取模恒等于 ccc 的 ddd 次方，即 Cd≡Me(mod n)C^d ≡ M^e (mod \ n)Cd≡Me(mod n) 一定成立，所以 M=Cdmod nM = C^d mod \ nM=Cdmod n。代入 C,d,nC,d,nC,d,n 的值，得到 M=78634117mod 9991=6M = 7863^{4117} mod \ 9991 = 6M=78634117mod 9991=6。

所以Bob就从密文 CCC 和私钥 (d,n)=(4117,9991)(d,n)=(4117, 9991)(d,n)=(4117,9991) 知道了加密之前的原文 M=6M=6M=6。

在整个通信过程Alice只用到了公钥 (e,n)(e,n)(e,n) 进行加密，Bob只用到了私钥 (d,n)(d,n)(d,n) 解密，没有任何关于秘钥的传递，只有加密后的密文 CCC 有可能在通信中被窃听到。

2.3 为什么RSA可以保证加密通信不被破解？

回顾上面的加密过程，我们用到了六个变量： p,q,n,φ(n),e,dp,q,n,\varphi(n),e,dp,q,n,φ(n),e,d，其中只有公钥 (e,n)(e,n)(e,n) 是公开的。想要破解密文，只要知道私钥 (d,n)(d,n)(d,n)，计算 M=Cdmod nM = C^d mod \ nM=Cdmod n 就可以破解 RSA\textit{RSA}RSA 算法。

那么，有没有可能在已知公钥 (e,n)(e,n)(e,n) 的情况下，推导出私钥 (d,n)(d,n)(d,n)？

根据 RSA\textit{RSA}RSA 构造的规则（见上述 RSA\textit{RSA}RSA 加密过程1-6步），可以得到以下信息：

  1. 因为公钥中已知 nnn，只要计算出 ddd，就能得到私钥。

  2. ed≡1(modφ(n))ed ≡ 1 (mod \varphi(n))ed≡1(modφ(n))，需要知道 eee 和 φ(n)\varphi(n)φ(n) 的值来求出 ddd。因为在公钥中已知 eee，所以只要求出 φ(n)\varphi(n)φ(n) 的值。

  3. φ(n)=(p−1)(q−1)\varphi(n)=(p-1)(q-1)φ(n)=(p−1)(q−1)，要求出 φ(n)\varphi(n)φ(n) 的值，需要求出 p,qp,qp,q 的值。

  4. n=p×qn=p \times qn=p×q，想要求出 p,qp,qp,q 的值，必须对 nnn 做因数分解。

结论：如果 nnn 可以被因数分解，ddd 就可以沿着4，3，2，1步骤推出，也就意味着私钥被破解。

但是大整数的质因数分解[4]，是一件非常困难的事情。目前，除了暴力破解，还没有发现别的有效方法。
