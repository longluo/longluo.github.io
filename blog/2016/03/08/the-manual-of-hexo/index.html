еҰӮдҪ•дҪҝз”ЁHexoе»әз«ӢдёҖдёӘиҮӘе·ұзҡ„дёӘдәәBlogеҸҠдё»йЎөзҫҺеҢ–пјҹ

еҸ‘иЎЁдәҺ 2016-03-08 жӣҙж–°дәҺ 2019-08-22 еҲҶзұ»дәҺ Techniques йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ
жң¬ж–Үеӯ—ж•°пјҡ 3.5k йҳ…иҜ»ж—¶й•ҝ ≈ 3 еҲҶй’ҹ

By Long Luo

жҲ‘зҡ„еҘізҘһд№ӢдёҖ@жҒўеӨҚеҗғзҙ зҡ„Fе°Ҹе§җжӣҫеҶҷдәҶдёҖзҜҮж–Үз« дёәдәҶдҝқжҢҒйҖјж јпјҢдёҚиҰҒеҒңжӯўеҶҷдҪңпјҢжҲ‘ж·ұд»Ҙдёәз„¶гҖӮ

дёӘдәәд№ҹжҳҜдёӘйқһеёёе–ңж¬ўиЈ…йҖјзҡ„дәәгҖӮе°ҸеӯҰж—¶еҸ‘зҺ°иҮӘе·ұеңЁж–Үеӯ—ж–№йқўжңүдёҖзӮ№зӮ№еӨ©иөӢж—¶пјҢе°ұе–ңж¬ўдёҠдәҶйҳ…иҜ»е’Ңз Ғеӯ—гҖӮдёҠеӯҰж—¶еҶҷеңЁж—Ҙи®°жң¬йҮҢпјҢеҗҺжқҘжҺҘи§ҰдәҶдә’иҒ”зҪ‘пјҢе°ұеҶҷеңЁдәҶзҪ‘з»ңеҚҡе®ўйҮҢпјҢж–ӯж–ӯз»ӯз»ӯеңЁж–°жөӘпјҢзҪ‘жҳ“пјҢQQз©әй—ҙзӯүйғҪз•ҷдёӢдәҶдёҖдәӣж–Үеӯ—з—•иҝ№гҖӮ

е·ҘдҪңд№ӢеҗҺпјҢжҲҗдәҶдёҖеҗҚзЁӢеәҸе‘ҳпјҢжңүж—¶еҖҷжҹҘиө„ж–ҷеҸ‘зҺ°еӨ§зҘһд»¬йғҪжңүиҮӘе·ұзҡ„дёӘдәәзҪ‘з«ҷж—¶пјҢж„ҹи§үйҖјж јеҘҪй«ҳгҖӮдҪңдёәдёҖеҗҚиҝҪжұӮйҖјж јзҡ„дёүжөҒзЁӢеәҸе‘ҳпјҢдәҺжҳҜд№ҺпјҢд№°з©әй—ҙпјҢжҹҘиө„ж–ҷпјҢжҠҳи…ҫпјҢз»ҲдәҺд№ҹжҗӯе»әдәҶдёҖдёӘеұһдәҺиҮӘе·ұзҡ„зҪ‘з»ңе°ҸеұӢгҖӮ

GitHubдҪңдёәзЁӢеәҸе‘ҳз•ҢдёӯжңҖеӨ§зҡ„еҗҢжҖ§дәӨеҸӢзҪ‘з«ҷпјҢйҖјж јдёҚзҹҘйҒ“й«ҳеҲ°е“ӘйҮҢеҺ»дәҶпјҒжҲ‘еҫҲеҝ«з§»жғ…еҲ«жҒӢпјҢеҹәдәҺGithub Pagesе»әдәҶдёҖдёӘдёӘдәәзҪ‘з«ҷпјҢжңҖејҖе§ӢжҳҜз”ЁеҹәдәҺJekyllзҡ„Octopressжҗӯе»әзҡ„гҖӮ

JekyllжҳҜеҹәдәҺPythonзҡ„пјҢи®ҝй—®иө·жқҘеҫҲж…ўпјҢиҖҢдё”дёҚеӨҹз®ҖжҙҒгҖӮдәҺжҳҜд№ҺпјҢеҹәдәҺNodeJSзҡ„HexoиҝҷдёӘйҖјж јжһҒй«ҳзҡ„зЁӢеәҸзҢҝеҶҷдҪңж–№ејҸиҝӣе…ҘдәҶжҲ‘зҡ„и§ҶзәҝпјҢдәҺжҳҜиҠұдәҶеҮ дёӘе°Ҹж—¶д»ҺOctopressиҝҒз§»еҲ°дәҶHexoгҖӮ

йӮЈд№ҲеҰӮдҪ•дҪҝз”ЁHexoе»әз«ӢдёҖдёӘеұһдәҺиҮӘе·ұзҡ„зҪ‘з«ҷе‘ўпјҹ

дёҖгҖҒ жіЁеҶҢGitHub

йҰ–е…ҲжҲ‘д»¬йңҖиҰҒеҒҡзҡ„жҳҜеҺ»GitHubжіЁеҶҢдёҖдёӘеёҗеҸ·пјҢе№¶еҲӣе»әдёҖдёӘRepositoryд»“еә“гҖӮиҝҷйҮҢйңҖиҰҒжіЁж„Ҹзҡ„жҳҜд»“еә“зҡ„еҗҚз§°еҝ…йЎ»жҳҜз”ЁжҲ·еҗҚ.github.ioпјҢжүҚиғҪдҪҝз”ЁGithub Pages.

дәҢгҖҒй…ҚзҪ®SSH

дёӢиҪҪе№¶е®үиЈ…GitгҖӮ

1. и®ҫзҪ®user nameе’Ңemailпјҡ

$ git config --global user.name "дҪ зҡ„GitHubз”ЁжҲ·еҗҚ"
$ git config --global user.email "дҪ зҡ„GitHubжіЁеҶҢйӮ®з®ұ"

2. з”ҹжҲҗsshеҜҶй’Ҙ:

иҫ“е…ҘдёӢйқўе‘Ҫд»Ө

$ ssh-keygen -t rsa -C "дҪ зҡ„GitHubжіЁеҶҢйӮ®з®ұ"

дёҖиҲ¬жғ…еҶөдёӢжҳҜдёҚйңҖиҰҒеҜҶз Ғзҡ„пјҢжүҖд»ҘпјҢжҺҘдёӢжқҘзӣҙжҺҘеӣһиҪҰе°ұеҘҪгҖӮ

жӯӨж—¶пјҢеңЁз”ЁжҲ·ж–Үд»¶еӨ№дёӢе°ұдјҡжңүдёҖдёӘж–°зҡ„ж–Үд»¶еӨ№.sshпјҢйҮҢйқўжңүеҲҡеҲҡеҲӣе»әзҡ„sshеҜҶй’Ҙж–Үд»¶id_rsaе’Ңid_rsa.pubгҖӮ

жіЁпјҡid_rsaж–Үд»¶жҳҜз§Ғй’ҘпјҢиҰҒеҰҘе–„дҝқз®ЎпјҢid_rsa.pubжҳҜе…¬й’Ҙж–Үд»¶гҖӮ

3. ж·»еҠ е…¬й’ҘеҲ°githubпјҡ

зӮ№еҮ»з”ЁжҲ·еӨҙеғҸпјҢиҝӣе…ҘSettings(и®ҫзҪ®)йҖүйЎ№гҖӮеңЁз”ЁжҲ·и®ҫзҪ®ж ҸпјҢзӮ№еҮ»SSH and GPG keysйҖүйЎ№пјҢз„¶еҗҺзӮ№еҮ»New SSH key(ж–°е»әSSH)жҢүй’®пјӣ

е°Ҷid_rsa.pubдёӯзҡ„еҶ…е®№еӨҚеҲ¶еҲ° Key ж–Үжң¬жЎҶдёӯпјҢз„¶еҗҺзӮ№еҮ»Add SSH key(ж·»еҠ SSH)жҢүй’®гҖӮ

4. жөӢиҜ•SSHпјҡ

$ ssh -T git@github.com

жҺҘдёӢжқҘдјҡеҮәжқҘдёӢйқўзҡ„зЎ®и®ӨдҝЎжҒҜпјҡ

The authenticity of host 'github.com (207.97.227.239)' can't be established. 
RSA key fingerprint is 17:24:ac:a5:76:28:24:36:62:1b:36:4d:eb:df:a6:45.
Are you sure you want to continue connecting (yes/no)?

иҫ“е…ҘyesеҗҺеӣһиҪҰгҖӮ

з„¶еҗҺжҳҫзӨәеҰӮдёӢдҝЎжҒҜеҲҷOK(е…¶дёӯзҡ„SeayXuжҳҜз”ЁжҲ·еҗҚ)гҖӮ

Hi longluo! You've successfully authenticated, 
but GitHub does not provide shell access.

д»ҘдёҠжҳҜеҮҶеӨҮе·ҘдҪңгҖӮ

дёүгҖҒеҲӣе»әжң¬ең°д»“еә“

1. ж–°е»әд»“еә“ж–Үд»¶еӨ№

иҝҷйҮҢе°ұеҸ–еҗҚдёәblogгҖӮ

$ mkdir blog

2. еҲқе§ӢеҢ–д»“еә“

иҝӣе…ҘеҲ°blogж–Үд»¶еӨ№пјҢжү§иЎҢеҲқе§ӢеҢ–е‘Ҫд»ӨгҖӮ

$ cd blog # еҲҮжҚўеҲ°blogзӣ®еҪ•
$ git init # еҲқе§ӢеҢ–gitд»“еә“

жіЁпјҡеҲқе§ӢеҢ–иҰҒеңЁHexoе®үиЈ…д№ӢеҗҺжү§иЎҢпјҢеӣ дёәеңЁhexoеҲқе§ӢеҢ–зҡ„ж—¶еҖҷдјҡд»ҺgithubдёҠе…ӢйҡҶд»Јз ҒпјҢдјҡеҲӣе»ә.gitж–Үд»¶еӨ№гҖӮ

еҰӮжһңеңЁжӯӨд№ӢеүҚеҲқе§ӢеҢ–д»“еә“дјҡе°ҶеҺҹжңүзҡ„д»“еә“дҝЎжҒҜиҰҶзӣ–жҺүпјҢжңҖеҗҺд№ҹдјҡеҲ йҷӨгҖӮ

еӣӣгҖҒHexoеҲқе§ӢеҢ–

1. е®үиЈ…Hexo

HexoжҳҜеҹәдәҺNodeJSпјҢжүҖд»ҘйңҖиҰҒе…Ҳе®үиЈ…NodeJSгҖӮ

$ npm install -g hexo-cli

2. еҲқе§ӢеҢ–жЎҶжһ¶

$ hexo init <yourFolder>
$ cd <yourFolder>

3. е®үиЈ…дҫқиө–

$ npm install

4. еҲқе§ӢеҢ–е®ҢжҲҗеӨ§жҰӮзҡ„зӣ®еҪ•пјҡ

.
в”ңв”Җв”Җ _config.yml //зҪ‘з«ҷзҡ„`й…ҚзҪ®`дҝЎжҒҜпјҢжӮЁеҸҜд»ҘеңЁжӯӨй…ҚзҪ®еӨ§йғЁеҲҶзҡ„еҸӮж•°гҖӮ
в”ңв”Җв”Җ package.json
в”ңв”Җв”Җ db.json // jsonж јејҸзҡ„йқҷжҖҒеёёйҮҸж•°жҚ®еә“    
в”ңв”Җв”Җ node_modules // Hexoзҡ„еҠҹиғҪJavaScriptж–Үд»¶
в”ңв”Җв”Җ public // з”ҹжҲҗйқҷжҖҒзҪ‘йЎөж–Үд»¶
в”ңв”Җв”Җ scaffolds   //жЁЎзүҲж–Үд»¶еӨ№гҖӮеҪ“жӮЁж–°е»әж–Үз« ж—¶пјҢHexoдјҡж №жҚ®scaffoldжқҘе»әз«Ӣж–Үд»¶гҖӮ
в”ңв”Җв”Җ source     //иө„жәҗж–Үд»¶еӨ№жҳҜеӯҳж”ҫз”ЁжҲ·иө„жәҗзҡ„ең°ж–№гҖӮ
|   в”ңв”Җв”Җ _drafts // иҚүзЁҝж–Үд»¶еӨ№
|   в””в”Җв”Җ _posts // ж–Үз« ж–Үд»¶еӨ№
в””в”Җв”Җ themes     //дё»йўҳж–Үд»¶еӨ№гҖӮHexoдјҡж №жҚ®дё»йўҳжқҘз”ҹжҲҗйқҷжҖҒйЎөйқўгҖӮ

5. ж–°е»әж–Үз« пјҲеҲӣе»әдёҖдёӘHello Worldпјү

$ hexo new <title> for example: "Hello World"

еңЁ/source/_postйҮҢж·»еҠ hello-world.mdж–Үд»¶пјҢд№ӢеҗҺж–°е»әзҡ„ж–Үз« йғҪе°Ҷеӯҳж”ҫеңЁжӯӨзӣ®еҪ•дёӢгҖӮ

еҰӮжһңиҰҒеҲ йҷӨпјҢзӣҙжҺҘеңЁжӯӨж–Үд»¶еӨ№дёӢеҲ йҷӨеҜ№еә”зҡ„ж–Үд»¶еҚіеҸҜгҖӮ

6. з”ҹжҲҗзҪ‘з«ҷ

$ hexo generate

жӯӨж—¶дјҡе°Ҷ/sourceзҡ„.mdж–Үд»¶з”ҹжҲҗеҲ°/publicдёӯпјҢеҪўжҲҗзҪ‘з«ҷзҡ„йқҷжҖҒж–Үд»¶гҖӮ

7. жңҚеҠЎеҷЁ

$ hexo server -p 3000

иҫ“е…Ҙhttp://localhost:3000еҚіеҸҜжҹҘзңӢзҪ‘з«ҷгҖӮ

8. йғЁзҪІзҪ‘з«ҷ

$ hexo deploy

иҜҘж“ҚдҪңдјҡе°Ҷhexoз”ҹжҲҗзҡ„йқҷжҖҒеҶ…е®№йғЁзҪІеҲ°й…ҚзҪ®зҡ„д»“еә“дёӯпјҢиҜ·зңӢдёӢйқўд»Ӣз»ҚгҖӮ

йғЁзҪІзҪ‘з«ҷд№ӢеүҚйңҖиҰҒз”ҹжҲҗйқҷжҖҒж–Үд»¶пјҢеҚіеҸҜд»Ҙз”Ё$ hexo generate -dзӣҙжҺҘз”ҹжҲҗе№¶йғЁзҪІгҖӮ

жӯӨж—¶йңҖиҰҒеңЁ_config.ymlдёӯй…ҚзҪ®дҪ жүҖиҰҒйғЁзҪІзҡ„з«ҷзӮ№пјҡ

## Docs: http://hexo.io/docs/deployment.html
deploy:
    type: git
    repo: git@github.com:YourRepository.git
    branch: master

еҰӮжһңжІЎжңүж„ҸеӨ–пјҢйғЁзҪІе°ұжҲҗеҠҹдәҶпјҢеҸҜд»Ҙжү“ејҖhttp://<з”ЁжҲ·еҗҚ>.github.ioжҹҘзңӢгҖӮ

дә”гҖҒдҪҝз”ЁNextдё»йўҳи®©з«ҷзӮ№жӣҙй…·зӮ«

1. дҪҝз”ЁNexT Theme

$ cd your-hexo-site
$ git clone https://github.com/iissnan/hexo-theme-next themes/next

д»ҺNextзҡ„Gihubд»“еә“дёӯиҺ·еҸ–жңҖж–°зүҲжң¬гҖӮ

2. Hexo NexTдё»йўҳзҡ„ж–ҮжЎЈз»“жһ„

/languages   #з”ЁжқҘй…ҚзҪ®еӣҪйҷ…еҢ–иҜӯиЁҖзүҲжң¬пјҢйҮҢиҫ№еҢ…еҗ«еҗ„з§ҚдёӘй…ҚзҪ®еғҸзҡ„ж–Үжң¬зҝ»иҜ‘гҖӮ
/layout      #д»Ҙswigж–Үд»¶жқҘе®ҡд№үеҗ„з§Қеҗ«жңүй…ҚзҪ®дҝЎжҒҜи°ғз”Ёзҡ„еёғеұҖ
/scripts     #дёҖдәӣJSи„ҡжң¬
/source    
    /css      #з”ЁжқҘдҝ®ж”№иҮӘе®ҡд№үж ·ејҸпјҢйңҖиҰҒжҺҢжҸЎдёҖе®ҡзҡ„cssиҜӯжі•гҖӮ
    /fonts    #е®ҡд№үеӯ—дҪ“ж–Үд»¶пјҢеҸҜд»Ҙдҝ®ж”№еҚҡе®ўеӯ—дҪ“
    /images   #дёҖдәӣsvgеӣҫзүҮ
    /js       #дёҖдәӣjsи„ҡжң¬
    /vendors  
/404.html #иҮӘе®ҡд№үзҡ„е…¬зӣҠ404йЎөйқў
/test        #з”ЁдәҺжөӢиҜ•

3. еҗҜз”ЁNexTдё»йўҳ

йңҖиҰҒдҝ®ж”№/root/_config.ymlй…ҚзҪ®йЎ№themeпјҡ

# Extensions
## Plugins: http://hexo.io/plugins/
## Themes: http://hexo.io/themes/

4. йӘҢиҜҒжҳҜеҗҰеҗҜз”Ё

$ hexo s --debug

и®ҝй—®http://localhost:4000пјҢзЎ®дҝқз«ҷзӮ№жӯЈзЎ®иҝҗиЎҢгҖӮпјҲжӯӨе‘Ҫд»ӨеҸҜд»ҘеҒҡе№іж—¶йў„и§Ҳз”Ёпјү

е…ӯгҖҒжҖ»з»“

йҖҡиҝҮд»ҘдёҠжӯҘйӘӨпјҢжҲ‘д»¬е°ұжҲҗеҠҹзҡ„е®һзҺ°дәҶеңЁGithubдёҠе»әз«ӢдәҶдёҖдёӘй«ҳйҖјж јзҡ„HexoдёӘдәәеҚҡе®ўз«ҷзӮ№гҖӮ
