[LeetCode][372. Super Pow] 2 Approaches: Brute Force and Binary Exponentiation

发表于 2021-12-05 更新于 2023-11-14 分类于 Data Structures and Algorithms Waline： 阅读次数： 本文字数： 1.7k 阅读时长 ≈ 2 分钟

By Long Luo

This article is the solution 2 Approaches: Brute Force and Binary Exponentiation of Problem 372. Super Pow.

Intuition

This problem is to find a integer raised to the power a very large number whose length may be 200200200 or more.

Brute Froce

We multiply aaa to itself bbb times. That is, ab=a×a⋯×a⏟ba^b = \underbrace{a \times a \dots \times a}_bab=ba×a⋯×a​​.

We can write such code easily.

Analysis

  • Time Complexity: O(10mbi)O(10^mb_i)O(10mbi​), mmm is the length of array b.
  • Space Complexity: O(1)O(1)O(1)

Obiviously, it will exceed time limit, so we have to find a more efficiently algorithm.

Binary Exponentiation

Recall the Fast Power Algorithm: Binary Exponentiation, we develop a fast power algorithm, so we can use it here directly.

We didn’t need to change the method of fast power.

Analysis

  • Time Complexity: O(∑i=0m−1log⁡bi)O(\sum\limits_{i=0}^{m-1} \log b_i)O(i=0∑m−1​logbi​), mmm is the length of array bbb.
  • Space Complexity: O(1)O(1)O(1)
