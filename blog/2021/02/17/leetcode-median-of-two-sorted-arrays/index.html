方法二：合并数组（优化空间）

思路与算法：

方法一需要新建一个长度为 len=m+nlen = m + nlen=m+n 的数组，但实际上我们并不需要将两个数组真的合并，我们只需要找到中位数在哪里就可以了。

同方法一，用一个循环计数，判断当前是否已经到了中位数的位置，到了就结束循环，但是需要考虑一些边界情况：

  1. 总长度为奇数；
    • 需要知道第 len+12\frac{len + 1}{2}2len+1​ 个数，遍历区间 [0,len2][0, \frac{len}{2}][0,2len​] 次，返回最后一次遍历的结果。
  2. 总长度为偶数；
    • 需要知道第 len2\frac{len}{2}2len​ 和 len2+1\frac{len}{2} + 12len​+1 个数，遍历区间 [0,len2][0, \frac{len}{2}][0,2len​] 次，返回最后一次和上一次遍历的结果。
  3. nums1nums1nums1 或者 nums2nums2nums2 遍历完之后；
  4. nums1nums1nums1 和 nums2nums2nums2 都没有遍历完。

我们需要用两个变量 leftleftleft 和 rightrightright，rightrightright 保存当前循环的结果，在每次循环开始前将rightrightright的值赋给 leftleftleft，这样在最后一次循环的时候，leftleftleft 将得到 rightrightright 的值，也就是上一次循环的结果，接下来 rightrightright 更新为最后一次的结果。

使用 222 个指针分别指向当前指向 nums1nums1nums1 数组和 nums2nums2nums2 数组的位置:

  1. 如果 p<mp < mp<m 且 nums1[p]≤nums2[q]nums1[p] \le nums2[q]nums1[p]≤nums2[q]，那么就可以后移了，p++p++p++；

  2. 如果 q≥nq \ge nq≥n，说明此时 nums2nums2nums2 已经没有数字了，p < m && (q \ge n || nums1[p] \le nums2[q])。
